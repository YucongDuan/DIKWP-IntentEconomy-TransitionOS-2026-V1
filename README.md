# DIKWP IntentEconomy TransitionOS 2026 V1

A standalone, offline-first prototype for the transition from attention economy to intention economy.

## Core thesis

The next economy should not optimize only clicks, impressions, dwell time, and behavioral prediction. It should organize user-declared intent, agent mandate, evidence-backed offers, auditable outcomes, consent, revocation, benefit sharing, and anti-manipulation governance.

## Quick start

Open `index.html` in a modern browser. No server, database, account, API key, ad network, payment API, or network is required.

Optional CLI:

```bash
python tools/run_intent_transition.py --asset-id AE-003 --out outputs
```

## What it implements

- Content/Data DIKWP Cards
- Stakeholder Intent Space
- Intent Contract and Agent Mandate
- Content-Intent Projection Matrix
- IntentValue Scorecard
- Attention-to-Intention metric map
- Action Tickets for hidden sponsorship, privacy gaps, purpose drift, and manipulation
- Transition Passport export

## Boundary

This is not an ad platform, not a behavioral targeting system, not a payment tool, not a transaction executor, and not a legal/compliance certification. It uses synthetic data only.
